# API para Trabajar con PDFs

Una API completa en Flask para manipular, firmar y extraer información de archivos PDF.

## Características

- ✅ Agregar imágenes a PDFs en coordenadas específicas
- ✅ Firmar PDFs con certificados digitales .pfx
- ✅ Extraer texto y metadatos de PDFs
- ✅ Buscar texto dentro de PDFs
- ✅ Verificar firmas digitales
- ✅ Crear nuevos PDFs con imágenes
